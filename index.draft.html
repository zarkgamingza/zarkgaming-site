How to join (quick)

  1. Open your game's Server Browser (or Steam server browser).
  2. Paste the server browser filter string from the cluster card into the search box.
  3. Subscribe to the required Steam Workshop mods, then restart the game if needed.
  4. If a map requires branch switching, follow the pinned Discord guide or ask in #support.

Need help? Click Join Discord and ping an admin in #support.
